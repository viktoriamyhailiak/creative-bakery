# Creative Bakery Landing Page

Creative Bakery Landing Page is a responsive landing page designed to showcase a bakery’s offerings, menu, and services.
The page follows a Figma design and is implemented using BEM methodology and SCSS for modular, maintainable, and scalable styling.

The layout uses CSS Grid and Flexbox to create a clean, visually appealing structure, optimized for all devices.

Live demo: [Creative Bakery Landing Page](https://viktoriamyhailiak.github.io/creative-bakery/)

Design reference (Figma): [Creative Bakery Figma](https://www.figma.com/file/dY3izAm0Vspsmra4lQWQIP/Bakerlab-FE-students?node-id=0%3A1)

# Technologies Used

HTML5 – semantic structure of the landing page

SCSS (Sass) – modular and maintainable styling

BEM (Block-Element-Modifier) – scalable CSS architecture

CSS Grid & Flexbox – responsive layout and alignment

Responsive Design – optimized for:

- Design: 1440px

- Desktop: 1280px

- Tablet: 640px

- Mobile: >320px

# Setup & Installation

Follow these steps to run the project locally:

1. Clone the repository
git clone https://github.com/<viktoriamyhailiak>/creative-bakery.git,
cd creative-bakery

2. Install dependencies
npm install or yarn install

3. Run the project locally
npm start or yarn start
